# Notepad++ Syntax Highlighter

## Installation

1. Download `burroughs-algol.xml`
2. Open Notepad++
3. Go to **Language → User Defined Language → Define your language...**
4. Click **Import** and select `burroughs-algol.xml`
5. Restart Notepad++
6. Associate with `.alg` or `.algol` file extensions (automatically configured)

## Features

- Case-insensitive keyword matching
- Core language keywords (ARRAY, POINTER, BEGIN, END, etc.)
- 500+ built-in functions organized by category
- Logical operators (AND, OR, NOT, EQV, etc.)
- Block comments: `COMMENT...;`
- Line comments: `%`
- String literals: `"..."`
- Number highlighting

## Color Scheme

- **Blue (bold)**: Core keywords
- **Purple (bold)**: Logical operators
- **Dark blue**: Built-in functions
- **Green**: Comments
- **Red**: Numbers and operators
- **Gray**: String literals
