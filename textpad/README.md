# TextPad Syntax Highlighter

## Installation

1. Download `ALGOL.syn`
2. Copy to TextPad syntax directory:
   - Windows: `C:\Program Files\TextPad\Samples`
   - Or via TextPad: Configure → Preferences → Document Classes → Add
3. Associate with `.alg` or `.algol` file extensions

## Features

- 46 core keywords (ARRAY, POINTER, INTEGER, REPLACE, etc.)
- 500+ built-in functions (database, math, I/O, transaction control)
- Preprocessor directives ($INCLUDE, $CODE, $SET, etc.)
- Comment styles: `COMMENT...;` and `%` inline
- Case-insensitive matching
